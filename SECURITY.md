# Security Policy

## Supported Versions

This project is a client-side web application hosted on GitHub Pages.
Only the latest deployed version is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- **Do NOT** open a public GitHub issue
- **Do NOT** disclose the vulnerability publicly

**Email:** umangdarji1999@gmail.com 
**Repository:** https://github.com/UmangDarji/bill-split

Provide as much detail as possible, including:
- Description of the issue
- Steps to reproduce
- Potential impact

## Disclosure Policy

We will:
- Acknowledge receipt of your report
- Investigate the issue promptly
- Fix the issue where appropriate
- Credit the reporter if desired

Thank you for helping keep this project secure.
